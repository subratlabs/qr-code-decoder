## QR Code Decoder

A Python application that decodes and reads QR codes from image files using the OpenCV library.

## Features

Decode text or URLs from any QR code image
Supports multiple image formats (.png, .jpg, .jpeg)
Prints the hidden data instantly to the console

## Technologies Used

Python
OpenCV Library (cv2)

## How to Run

Install the required libraries.
Place your QR code image in the project folder.
Run the Python file.
View the decoded text or URL in your terminal.
